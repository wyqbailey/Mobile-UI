添加自定义样式

主页

JQM自身的jquery.mobile-1.0b1.css样式表默认提供了所有组件的样式，但缺点是关联性非常强，在不熟悉样式表的情况修改会影响到其他控件，建议采用复写或优先级的方式来定义新的样式。

复写样式表

和PC端一样，在头部引入外部css文件即可。但需要注意的是，如果你的项目中启用了异步模式（ajaxEnabled: true，默认关闭），那么引入css文件时必须将link标签放置在body内的第一个page组件内。

<link rel="stylesheet" href="../nresources/css/style.css"> 默认样式表

<link rel="stylesheet" href="../nresources/css/style.css">用于复写组件和新组件的样式表

优先级问题

在每个页面dom加载完成之后，框架会对页面上的控件进行一系列初始化处理，在处理过程中会为控件追加一些class或者样式，这些class或行内样式的优先级都较高。如果你想复写控件样式或复写控件的某个表现，请使用!important规则提高自定义样式的优先级。
