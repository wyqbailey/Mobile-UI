引入js文件

和PC端一样，在头部引入外部js文件即可。但需要注意的是，如果你的项目中启用了异步模式（ajaxEnabled: true，默认关闭），那么引入js文件时必须将script标签放置在body内的第一个page组件内。

内联js代码

和PC端一样，你可以在页面头部编写内联js代码。但需要注意的是，如果你的项目中启用了异步模式（ajaxEnabled: true，默认关闭），那么这些js代码必须放置在body内的第一个page组件内。

js事件差异

移动设备由于其硬件特性，一般都没有鼠标事件（onmouseover，onmouseup，onmousemove等），取而代之的是手势事件（swipe，swipeLeft，swipeRight等），有关事件的详细说明，请见技术文档。
