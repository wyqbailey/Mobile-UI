主页

引入文件

    <link rel="stylesheet" href="../nresources/jqthemes/blue/jquery.mobile-1.0b1.css"/>
    <script type="text/javascript" src="../njs/jquery.js"></script>
    <script type="text/javascript" src="../njs/jquery.mobile-1.0b1.js"></script>
    <script type="text/javascript" src="../njs/iscroll.js"></script>

注意

先引入jquery，再引入jquery mobile；jquery版本必须是1.6以上。

iscroll.js在登录和闪屏页面不需要引用，只有在使用标题栏和工具栏固定定位的时候需要引用。
