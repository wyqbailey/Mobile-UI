创建一个html文档

    <!DOCTYPE html>
    <html>
        <head>
            <title>Page Title</title>
            <meta name="viewport" content="width=device-width, initial-scale=1">
        </head>
        <body>

            ...content goes here...

        </body>
    </html>

使用html5标准

DOCTYPE声明为<!DOCTYPE html>

控制页面缩放及viewport宽度

<meta name="viewport" content="width=device-width, initial-scale=1">

初始缩放比例为1（不缩放），viewpoint宽度为设备宽度。更多有关viewpoint的知识，请见：http://eip.si-tech.com.cn/viewLog.action?blogUserId=8898&logId=69205和http://learnthemobileweb.com/2009/07/mobile-meta-tags/
