配置控件

框架提供了丰富的控件（详见控件演示），每个控件可能根据不同的应用场景需要有不同的样式和交互行为，因为框架为每个控件提供了一些个性配置参数。下面以按钮控件为例来演示一下控件配置：

默认配置
<button type="button">按钮</button>
内联按钮： data-inline
<button data-inline="true" type="button">按钮</button>
按钮图标： data-icon
<button data-icon="delete" type="button">按钮</button>
图标位置： data-iconpos
<button data-icon="delete" data-iconpos="right" type="button">按钮</button>

控件配置方法

每个控件的配置参数、参数值和配置方法，详见控件演示或技术手册。
