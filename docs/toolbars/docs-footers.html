底部栏的html结构

底部栏和标题栏的html结构差不多，除了data-role="footer"属性不同。

<div  data-role="footer">
	<h4>Footer内容</h4> 
</div>

Footer 内容

底部栏的设置使用比较灵活，不会像header一样自动生成一些按钮之类,根据需要自行填充内容。

添加按钮

在footer中的任何有效链接都会被渲染成按钮，为了节省有限的空间，所有的按钮都是以内联样式存在，按钮默认只包含图标和文字，而没有内边距,如果需要设定内边距，请在footer中调用 class="ui-bar"。


    <div data-role="footer" class="ui-bar">
    <a href="index.html" data-role="button" data-icon="delete">Remove</a>
    <a href="index.html" data-role="button" data-icon="plus">Add</a>
    <a href="index.html" data-role="button" data-icon="arrow-u">Up</a>
    <a href="index.html" data-role="button" data-icon="arrow-d">Down</a>
    </div>

如果需要把几个按钮组合成一个按钮组，则使用包含 data-role="controlgroup" 和 data-type="horizontal"属性的容器将一组按钮包裹起来。按钮组的具体用法请见按钮类控件文档。
