固定的底部导航栏

主页

本页面展示了固定的底部导航栏效果。在页面下部的底部栏（footer）中嵌套了一个导航栏（navbar），当你点击其中的导航按钮进行页面切换时，底部导航栏的位置保持不变。

用法：在导航栏链接的3个页面中都是用同样的底部导航栏结构，如下：


   <div data-role="footer" data-id="foo1" data-position="fixed">
     <div data-role="navbar">
       <ul>
         <li><a href="footer-persist-a.html" class="ui-btn-active ui-state-persist">朋友</a></li>
         <li><a href="footer-persist-b.html">相册</a></li>
         <li><a href="footer-persist-c.html">邮件</a></li>
       </ul>
     </div><!-- /navbar -->
    </div><!-- /footer -->

注意：在底部栏（footer）上调用data-position="fixed" 的同时 ，在需要保持footer一致的页面中，设置footer的 data-id 值相同。 如果你想在转场过后，仍然保持footer中的激活按钮不变，则请为此按钮调用 ui-state-persist 和 ui-btn-active的class。
