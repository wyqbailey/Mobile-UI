标题栏的html结构

标题栏位于页面的顶部，通常包含了页面标题和居于标题的左右两侧的导航按钮，最多两个。

其中的标题一般用h1标签标记，不过h1-h6都是可以的。举个例子：如果一个页面包含多个page控件，则可以用h1标记主页，用h2标记次页。

<div  data-role="header"> 
	<h1>Page Title</h1> 
</div>

页面标题

添加按钮

标准的header配置中，在标题的两侧可以各放一个按钮控件，按钮控件的标签可以使用a，button或input任何一种，按钮控件的标签选择请见button markup。

按钮的默认位置

标题栏中的第一个按钮控件默认被定位在左边，第二个默认定位在右边。举个例子：

			
<div data-role="header" data-position="inline">
	<a href="index.html" data-icon="delete">删除</a>
	<h1>编辑联系人</h1>
	<a href="index.html" data-icon="check">保存</a>
</div>
删除

编辑联系人

保存

按钮的主题会默认继承标题栏的主题定义，当然也可以根据操作对象进行自定义，比如：“删除”按钮采用a主题，而“保存”按钮采用b主题。

			
<div data-role="header" data-position="inline">
	<a href="index.html" data-icon="delete">删除</a>
	<h1>编辑联系人</h1>
	<a href="index.html" data-icon="check" >保存</a>
</div>
删除

编辑联系人

保存

用class来控制按钮的位置

按钮的位置可以通过class来控制。比如你想把"设置"按钮放在右侧，那么你只需为此按钮调用“ui-btn-right”这个class即可。别忘了还有个 ui-btn-left .


<div data-role="header" data-position="inline" 
	<h1>Page Title</h1>
	<a href="index.html" data-icon="gear" class="ui-btn-right">设置</a>
</div>

设置

添加返回按钮

Mobile UI会在标题栏中自动包含一个“返回”按钮（在page控件中的顶级标题栏会包含，在内容区的其他标题栏不会包含），简化开发的复杂性，相当于是浏览器的返回按钮。你可以在page控件上中使用 data-add-back-btn="true" 属性来控制其可视性。

自定义”返回“按钮的文字

1、在page控件上中调用data-back-btn-text="自定义文字"属性;
2、js方法： $.mobile.page.prototype.options.backBtnText = "自定义文字";.

自定义标题栏

如果你不想采用Mobile UI为你提供的默认配置的标题栏，那么你可以在标题栏容器中用一个div容器包括你的自定义内容即可。
