Toolbar类型

在jQuery Mobile中，提供了两种标准类型的toolbar： Headers and Footers.

  • Header bar 作为页面标题，通常是每个mobile page中的第一个元素, 内容：页面标题、最多连个按钮
  • Footer bar通常是每个mobile page的最后一个元素，可使用比较自由的格式，常包含文字描述和按钮.

在header和footer中最常见的是水平导航和标签栏。jQuery Mobile 包含了一个nav bar widget，它让一个无序链接列表变成横向的按钮工具条

Toolbar定位设置

Header and footers能通过几种不同的方法被定位在页面上.默认情况下toolbars用"inline" 定位 模式. 在这种模式下headers and footer将以文档流（HTML行为）的形式按顺序显示，这样可以保证在所有的设备上都保持一致的体验。

"fixed"定位模式提供了一个方便的静态工具栏方法，而弥补了JavaScript的滚动弊端，当页面发生滚动的时候，toolbar的位置会动态地重新定位.

在任何时候，轻击屏幕将会切换fixed toolbar的可视性。在PC上开发时，鼠标的点击事件会触发可视性，在触摸设备上时tap轻击事件触发

用法：添加 data-position="fixed"属性到 toolbar容器中.

"fullscreen"定位模式和fixed模式相似，但是它的toolbar默认是隐藏的，只有当页面被点击的时候才显示。这种方式对诸如视频播放和照片浏览的页面很有用处，当你轻击屏幕的时候Toolbar才显示（注意：这时的toolbar会遮挡住一部分内容区域）。
