普通链接列表

最简单的列表组件其实就是包含data-role=”listview”的一个无序列表，每个列表项都含有一个链接。Jquery mobile会自动应用必要的样式，在每个列表项的右侧追加右箭头图标，使其宽度充满屏幕，这样对移动设备更友好。当你点击一个列表项时，实际上就是点击了一个链接，这会触发一个ajax请求，最终切换界面显示。一个简单的基本链接列表的代码如下：


<ul data-role="listview" data-theme="g">
	<li><a href="acura.html">Acura</a></li>
	<li><a href="audi.html">Audi</a></li>
	<li><a href="bmw.html">BMW</a></li>
</ul>

嵌套列表

给一个列表项嵌入一个ul或ol，就可以创建一个嵌套列表。当点击列表项时，jquery mobile会自动生成一个新的使用列表项名称作为页面标题的page组件来显示其子列表，这些动态的嵌套列表默认使用主题中的b皮肤（蓝色）以便表明他们是第二级导航。你可以嵌套若干级列表，所有嵌套关系生成的新的page组件都有框架自动控制。

如果想改变嵌套列表的样式，使用data-theme属性指定即可。

有序列表

也可以使用ol来创建有序列表组件，可用于如搜索结果或电影排行榜之类的地方。生成有序列表时，jquery mobile会尝试使用css为列表添加序号显示，如果不行就使用javascript来插入数字。

只读列表

列表组件也可以用来只显示只读的一些不具备交互性的信息，只读列表依然使用ul或ol创建，只是每个列表项内没有链接而已。只读列表默认使用c皮肤而且字体较小，这样好节省点空间。

按钮分离列表

假如一个列表项需要多个交互操作，就可以使用按钮分离列表，它的每个列表项可以提供两块交互区域：列表文字和右侧的小图标。要创建一个按钮分离列表，只需要在列表项内再追加一个链接，这样框架就会自动在两个链接中间加入一条分割线，将第二个链接变成一个纯图标的按钮，并将链接文字设置为按钮的title属性值以提供易用性支持。

你可以通过data-split-icon属性来设定列表项右侧的小图标，当然还可以通过data-split-theme属性来设置其主题皮肤。

列表分割符

任意一个列表项都可以通过追加data-role=”list-divider”属性转变成一个列表分隔符以便分组整理列表项。列表分隔符默认使用b皮肤以便与普通列表项区别开，但你也可以使用data-dividertheme属性来设定其他皮肤。

搜索栏

Jquery mobile提供了一个非常简单的办法给列表组件增加一个基于客户端的搜索栏。要实现这点，只需给列表组件追加一个data-fliter=”true”属性。这样框架就会给列表自动增加一个搜索栏，同时为其增加根据输入文字过滤列表项的行为。搜索栏的默认显示文字是“Fliter items…”，配置这段文字有两个办法：一是使用$.mobile.listview.prototype.options.filterPlaceholder这个配置项来定义默认显示的字符串；二是在列表组件上追加data-fliter-placeholder属性。默认情况下搜索栏继承列表组件的皮肤，当然你可以通过data-fliter-theme属性来配置其他皮肤。

文字排版和数量指示

框架为列表中常用的的文字排版（如标题、二级描述、数量指示等）提供了一些规约。

  • 要在列表项右侧添加一个数量指示器，只需要将数字用包含ui-li-count类的容器包裹。
  • 调整文字层级结构：使用标题元素来提高层架，使用p元素来降低层级。
  • 创建位于列表项右侧的补充信息可以使用包含ui-li-aside类的元素来包裹。

缩略图和图标

要在列表项左侧增加缩略图，直接在列表项加入一个图片并使其成为第一子元素即可。框架会自动将图片缩放成80像素的正方形。如果想使用图标，给图片增加一个ui-li-icon类即可。

内嵌列表

列表默认宽度铺满了整个屏幕。假设列表作为页面中某个区域的一部分，使用成有一定边距和圆角的内嵌列表在视觉上更合理。给列表组件增加data-inset=”true”即可做到这一点。

内嵌列表示例

更新列表

如果你为列表组件追加了列表项，你需要调用refresh方法来刷新列表显示，代码如：$('ul').listview('refresh');
