表单按钮

为了便于应用样式，框架会自动将button标签和部分类型input标签（submit、reset、button或image）转变成统一样式的按钮（使用div，span来封装模拟），无须添加data-role=”button”属性。原有的button类标签被保留在dom中，但会被隐藏。当点击可见的按钮时，会自动触发原始标签上的click事件。

<button>Button element</button> <input type="button" value="Input type=button"/> <input type="submit" value="Input type=submit"/> <input type="reset" value="Input type=reset"/> <input type="image" value="Input type=image"/>
