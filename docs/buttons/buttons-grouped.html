按钮组

有时你可能想让一组按钮组合在一起，形成一个控制模块或导航模块。要实现这种效果，只需将一组按钮使用一个包含data-role="controlgroup"的容器包裹起来:


<div data-role="controlgroup">
    <a href="index.html" data-role="button">Yes</a>
    <a href="index.html" data-role="button">No</a>
    <a href="index.html" data-role="button">Maybe</a>
</div>

默认按钮组是垂直排列的:

Yes No Maybe

水平按钮组

给包裹容器追加data-type="horizontal"属性，就可以使这组按钮水平排列，宽度根据按钮内容自适应。（注意，如果水平按钮组内按钮过多，可能导致按钮换行显示，影响效果）。


<div data-role="controlgroup" data-type="horizontal">
    <a href="index.html" data-role="button">Yes</a>
    <a href="index.html" data-role="button">No</a>
    <a href="index.html" data-role="button">Maybe</a>
</div>
Yes No Maybe
