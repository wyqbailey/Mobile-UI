给按钮增加图标

框架自带了一组在移动应用中经常被使用的图标。使用data-icon属性可以为按钮设定一个特定图标，例如:

<a href="index.html" data-role="button" data-icon="delete">Delete</a> Delete

图标位置

默认情况下，所有图标都被放置在按钮文字的左边。

Delete

你可以使用data-iconpos属性来设置使图标位于按钮文字的右边、上边或下边。例如:

<a href="index.html" data-role="button" data-icon="delete" data-iconpos="right">Delete</a> Delete <a href="index.html" data-role="button" data-icon="delete" data-iconpos="top">Delete</a> Delete <a href="index.html" data-role="button" data-icon="delete" data-iconpos="bottom">Delete</a> Delete

纯图标按钮

你还可以创建只有图标的按钮，只需要将data-iconpos设置成notext，举例:

<a href="index.html" data-role="button" data-icon="delete" data-iconpos="notext">Delete</a>
