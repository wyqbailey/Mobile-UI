创建dialog

给任何一个链向page组件或页面的链接上增加data-rel="dialog"属性，便可将目标page组件模拟成一个模态的dialog组件，框架会自动给dialog组件添加圆角、外边距和深色背景使其看起来弹出在界面上。

<a href="foo.html" data-rel="dialog">Open dialog</a>

Open dialog

切换效果

Dialog组件默认使用“pop”效果打开，但和page组件一样，你可以通过data-transition属性给他设定其他效果。但为了使其看上去像个弹出对话框，我们建议你使用“pop”，“slideup”或者“flip效果”。

<a href="foo.html" data-rel="dialog" data-transition="pop">Open dialog</a>
data-transition="pop" data-transition="slidedown" data-transition="flip"

关闭dialog

当点击dialog组件内的任意链接时，dialog组件会自动关闭，页面会切换到链接请求页面。如果想创建一个“取消”按钮，只需将链接指向触发dialog打开的页面，然后给链接添加data-rel=”back”属性，这样的链接在不支持js的设备中仍能正常跳转。

对于由javascript生成的链接，你可以将链接的href属性设置为“#”，然后添加data-rel=”back”属性。你还可以使用close方法来关闭dialog，像这样：$('.ui-dialog').dialog('close')。
