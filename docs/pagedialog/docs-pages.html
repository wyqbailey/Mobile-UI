page控件（视图）

Home

Page控件用来构造一个独立的显示界面（视图）。一个html文档中可以有多个page控件，也可仅包含一个page控件，但界面只会显示其中一个page，不会多个page同时显示。我们建议在开发中每个页面只包含一个page控件，如果page控件的结构非常简单，则可考虑多个page控件组合在一个页面（这样更节省带宽，界面之间的切换也更快）。Html文档加载完成后，默认显示的是文档顺序中第一个page控件，所以编写包含多个page控件的html文档时要注意顺序。

一个标准页面的html结构

Mobile UI页面的doctype必须使用html5标准；在头部引入css和js文件；使用data-role来定义一个page控件，page控件内可以放置任意html元素，但一般都使用如下所示的3段式构造：header、content、footer


    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    <link rel="stylesheet" href="http://code.jquery.com/mobile/1.0b1/jquery.mobile-1.0b1.min.css"
    />
    <script type="text/javascript" src="http://code.jquery.com/jquery-1.6.1.min.js"></script>
    <script type="text/javascript" src="http://code.jquery.com/mobile/1.0b1/jquery.mobile-1.0b1.min.js"></script>
    </head>
    <body>

    <div data-role="page">

    <div data-role="header">
    <h1>Page Title</h1>
    </div><!-- /header -->

    <div data-role="content">
    <p>Page content goes here.</p>
    </div><!-- /content -->

    <div data-role="footer">
    <h4>Page Footer</h4>
    </div><!-- /footer -->
    </div><!-- /page -->

    </body>
    </html>
